# Django Sticky Notes Application (Part 1)

This project is a simple **Sticky Notes web application** built using the Django framework.  
It allows users to **create**, **view**, **edit**, and **delete** notes, displaying them in a clean, user-friendly interface.

---

## Project Overview

The Sticky Notes Application demonstrates the fundamentals of Django web development, including:

- Setting up a Django project and app structure
- Creating models and migrations
- Using views and templates
- Applying CRUD (Create, Read, Update, Delete) operations
- Organizing HTML templates with the Django templating engine

---

## Features

- Add new notes with a title and content  
- View all existing notes  
- Edit/update notes  
- Delete notes  
- Notes are automatically ordered by creation date

---

## Technologies Used

- **Python 3.11+**
- **Django 5.2+**
- **SQLite3** (default database)
- **HTML / CSS** (for templates and styling)
- **VS Code** (recommended editor)
